🧩 Explicando o Desafio

Desenvolvi o Travelgram como parte de um desafio de front-end com o objetivo de praticar a construção de uma página estática utilizando apenas HTML e CSS.

A proposta era recriar um layout de perfil de viagens, focando principalmente em organização do código, estrutura semântica e estilização.

Durante o desenvolvimento, trabalhei com:

✔️ Estruturação em HTML

Busquei organizar o conteúdo de forma clara e semântica, representando um perfil de usuário com informações como localização, quantidade de países visitados e fotos.

✔️ Estilização com CSS

Utilizei Flexbox para alinhar os elementos na tela, além de ajustar espaçamentos, tipografia e imagens para chegar em um layout visualmente agradável.

✔️ Organização do Projeto

Estruturei os arquivos separando os estilos e os assets (imagens), mantendo o código limpo e fácil de entender.

🎯 Resultado

Ao final do desafio, consegui construir uma página simples, porém bem organizada e com um visual consistente. Esse projeto serviu como base para reforçar conceitos importantes de front-end e preparar o terreno para projetos mais avançados, como páginas responsivas e aplicações dinâmicas.
